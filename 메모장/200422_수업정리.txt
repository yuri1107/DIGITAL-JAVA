1. 배열 - (같은 의미의) 같은 자료형을 가진 변수들의 집합
자료형 배열명[];
자료형[] 배열명;
int arr1[], arr2; // arr2는 변수일 뿐 배열이 아님
int[] arr3, arr4; // arr3,4 모두 배열.
arr2 = new int [4]; // arr2라는 배열을 만드는데 이건 4칸짜리 배열이다.

- index : 0 ~ 배열크기-1
for(int i=0; i<arr2.length; i++) { }

2. 향상된 for문 - 배열(컬렉션)에 저장된 요소들을 읽어오는 용도로만 사용 가능. 

for(자료형 변수명 : 배열) {
	실행명;
}
