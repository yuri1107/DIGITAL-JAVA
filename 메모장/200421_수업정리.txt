1. 반복문
break : 반복문을 빠져나가는 역할, 무조건 조건문과 같이 쓰임.

continue : 스킵, for문에서는 증감연산식 위치로, while문에서는 조건식 위치(증감연산식이 없기때문에)로 이동.

while (조건식) {
	실행문;
}

이거를

초기화;
while (조건식) {
	실행문;
	증감연산식;
}

반복문 차이, 자주 쓰는 용도
for문 : 일정한 크기로 값이 증가하면서 반복될때 (규칙이 있을 때)
while문 : 주로 간단한

do while 문은 무조건 1번은 실행된다.
do {
	실행문;
while(조건식);
}

2. 배열
- (같은 의미와) 같은 자료형을 가진 변수들의 집합.
왜 ?
- 일반 변수 10개를 이용해서는 반복문을 사용할 수 없다.
- 효율적으로 변수들을 관리하기 위해서.

-선언방법
	1. 자료형 [] 배열명;
	2. 자료형 배열명 [];
- 배열은 선언 후 생성까지 해야함.
ex: int []arr5 = new int[5]; // int형 배열 arr5 를 선언하고, 안에는 int형 변수가 5개 들어있으며 인덱스는 0~ 4 까지 호출가능.
