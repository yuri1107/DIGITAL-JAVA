1. 자식 -> 부모 자동형변환
2. 부모 -> 자식 명시적 형변환

부모클래스이름 : P
자식클래스이름 : C

Pp = new P();
Cc = new C();
P p1 = c;
C c2 = (C)p; // ClassCastException 에러 발생. 
C c3 = (C)p1;


부모 -> 자식 명시적형변환은 항상 가능한것은 아님.

p1은 부모객체가 아닌 자식객체 c를 저장했다가 다시 c3에게 형변환해서 넘겨주기때문에 
부모 -> 자식 명시적 형변환 이지만 에러발생하지않는다.

하지만 p는 부모객체를 c2인 자식 객체에 저장하려 하기 때문에 
2부모 -> 자식 명시적 형변환 이지만 형변환이 불가능하여 에러 발생.
